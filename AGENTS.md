# Project Startup Instructions

Before doing work in this repository, read these files and keep their instructions in context:

1. `README.md`
2. `CLAUDE.md`

Treat `README.md` as the repo overview and setup guide.

Treat `CLAUDE.md` as the repo-specific mechanical intent, kinematics, key numbers, and gotchas for this 3D-printing project. Read it first each session and keep it current as decisions land.

The build loop, verification gates, and print conventions are all documented in `CLAUDE.md`. Run `make help` for the available targets.
